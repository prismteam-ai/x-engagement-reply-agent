# Docs

Supporting documentation for the X Engagement Reply Agent.

| File | What it is |
|------|------------|
| [`configuration.md`](./configuration.md) | Full schema for the code-managed config (`config/watchlist.yaml`, `config/settings.yaml`) and the prompt files (`prompts/**`). |
| [`deployment.md`](./deployment.md) | Agent summary (purpose, triggers, inputs, outputs, dependencies), required credentials, local run steps, the Docker dev stack, and the AWS production deployment. |
| [`reference-architecture.md`](./reference-architecture.md) | Pipeline map for the original investors-mcp workflow this agent extracts (provided reference). |

See also [`USAGE.md`](../USAGE.md) (quickstart + commands) at the repo root. The recorded
demo, the step-by-step demo walkthrough, and a captured dry-run sample are provided in the
pull request description.
